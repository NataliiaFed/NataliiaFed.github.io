Вітаю на сторінці-лендінгу мого телеграм-бота!

Тут буде міститись опис телеграм-бота, над яким я працюю в рамках навчання на курсі Python Developer від Robot Dreams. Попереденьо він буде включати такий обовязковий функціонал:

  • Видавати список доступних команд по запиту (“команди”, “/commands”)
  • Мати режим телефонної книги: зберігати, відображати, видаляти інформацію з іменами і номерами телефонів.
  • Має працювати незалежно для кожного окремого користувача, тобто не надавати інформацію із чужої телефонної книги іншим користувачам.
